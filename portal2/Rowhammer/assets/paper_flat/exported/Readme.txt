To compile, use the following settings in Crowbar:

QC input: file, set to QC, eg. portal2\Rowhammer\assets\paper_flat\exported\paper_bent.qc
Output to: work folder, set to content root, eg. portal2\Rowhammer\content
Game: Portal 2

Textures should be created with VTFEdit with normal and alpha formats set to DXT1.
